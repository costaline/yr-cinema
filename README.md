# yr Cinema

## feature list

- [x] pagination view list of films
- [x] user authentication
- [x] partial i18n (en, ru)
- [x] view selected film info
- [ ] add new film to db
- [ ] remove film from db
- [ ] user profile
  - [ ] change profile info
  - [ ] delete profile
- [ ] system-based dark mode
- [ ] adapted for mobile
